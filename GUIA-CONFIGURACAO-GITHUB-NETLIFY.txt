# Configuração rápida GitHub + Netlify

## Variáveis obrigatórias no Netlify
- GITHUB_TOKEN
- GITHUB_REPO=stepmanus-boop/Step-Gerencia
- GITHUB_BRANCH=main
- SESSION_SECRET

## API real Smartsheet
- SMARTSHEET_TOKEN
- SMARTSHEET_SHEET_NAME
ou
- SMARTSHEET_SHEET_ID

## Teste
1. Entrar com admin / admin123
2. Criar um usuário
3. Atualizar a página
4. Confirmar se o usuário continua salvo
5. Fazer login com o usuário criado

Se o GitHub estiver configurado, usuários e alertas ficam persistidos para todos os dispositivos.
